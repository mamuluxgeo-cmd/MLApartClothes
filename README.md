# MLApartClothes

სასტუმროს მაღაზიის ონლაინ კატალოგი + სამართავი პანელი + სალარო + Google Sheets backend.

## ფაილები

- `index.html` — მომხმარებლის საიტი
- `style.css` — მთავარი ვიზუალი
- `script.js` — მომხმარებლის საიტის ლოგიკა
- `admin.html` — სამართავი პანელი
- `admin.js` — პროდუქტის, შეკვეთის, სურვილების და ანალიტიკის მართვა
- `cashier.html` — სალაროს პანელი
- `cashier.js` — გაყიდვა, ფასდაკლება, ქეში/ტერმინალი, დღის დახურვა
- `config.js` — Apps Script Web
